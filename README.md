CREATED A WEB APPLICATION USING MERN(Mongo,Express,React,Node) STACK 
MongoDB was used for Database Management by storing data in a NoSQL format to achieve persistent storage for the user's notes.

Express was used for Web Framework/Routing by creating RESTful API endpoints to achieve communication between the frontend and the server.

React was used for Frontend Development by building reusable UI components to achieve a dynamic and responsive user interface.

Node.js was used for Server-Side Runtime by running JavaScript outside the browser to achieve a scalable backend environment.

Tailwind CSS & Daisy UI were used for Styling by applying utility-first CSS classes and pre-made components to achieve a professional and modern visual design quickly.

Mongoose was used for Object Data Modeling (ODM) by defining schemas and models for the note data to achieve strict data validation and easier database interactions.

Axios was used for HTTP Requests by fetching data from the backend API to achieve seamless integration of live data into the React UI.

Upstash (Redis) was used for Rate Limiting by tracking request frequency per user to achieve protection against API abuse and server overwhelming.

Render was used for Cloud Hosting by connecting to the GitHub repository and automating builds to achieve a live, publicly accessible application link .

Dotenv was used for Environment Variable Management by storing sensitive credentials like database URIs in an .env file to achieve secure code that hides private keysfrom public repositories
